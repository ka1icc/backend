# Создать 2 ветки в репозитории ka1icc/backend

Открой **Git Bash** и выполни по порядку.

## Если репозиторий ещё не клонирован

```bash
cd /c/Users/yulia/Desktop
git clone https://github.com/ka1icc/backend.git backend-repo
cd backend-repo
```

## Если backend-repo уже есть (папка уже существует)

Просто зайди в неё и выполняй блок ниже:

```bash
cd /c/Users/yulia/Desktop/backend-repo
```

## Создать две ветки и отправить на GitHub

```bash
# Актуальный main
git fetch origin
git checkout main
git pull origin main

# Ветка 1: 1-avito
git checkout -b 1-avito
git push -u origin 1-avito

# Ветка 2: 2-ivelum
git checkout main
git checkout -b 2-ivelum
git push -u origin 2-ivelum

# Вернуться на main
git checkout main
```

После этого на GitHub в репозитории будут ветки **main**, **1-avito** и **2-ivelum**. Ветки пока пустые (как main); код заданий добавляешь в каждую ветку отдельно.
